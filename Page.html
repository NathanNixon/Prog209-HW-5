    Grader    

This is an application to allow teachers to easily keep track of the grades they give for their students assignments.

It tracks the name of the student, the name of the class, the name of the assignment, and the grade when entered on the Grade Assignments page. It will then display that information in an unsorted table on the List Grades page.

It is notably superior to the last project in compiling the table, since I fixed the issue where running it multiple times would create duplicate entries.

Please note that Teal.css is a jQuery Mobile theme, autogenerated by this ThemeRoller to give us a data-theme='c' to use. It does not represent original code-work by our team, and does not need to be graded (though it is neccesary to download for the page to display properly). The files created by us are Page.html, Page.css, Page.js, EatenApple.png, and UneatenApple.png.

Have an apple, so you don't get hungry while grading.
